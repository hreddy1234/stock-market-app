# stock-market-app
The Regime Switching Volatility Predictor is a machine learning-based project developed to analyse and classify stock market conditions into High Risk and Low Risk regimes. Financial markets  highly dynamic and often shift between periods of stability and volatility, making it important to identify these changes accurately  better decision-making.
Model Used:
Logistic Regression
Type: Classification
Output:
0→ LOW risk
1→ HIGH risk

Why Logistic Regression?
In this project, we experimented with multiple machine learning models such as K-Nearest Neighbors (KNN). Random Forest, Support Vector Machine (SVM). XGBoost, and Logistic Regression to find the best performing model.
After evaluating all models, Logistic Regression provided the highest accuracy and more consistent results compared to others. It performed well for this binary classification problem (HIGH risk vs LOW risk) and was also simple, fast, and easy to interpret.
Therefore, Logistic Regression was selected as the final model for predicting stock risk.
